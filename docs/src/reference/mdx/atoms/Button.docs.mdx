## Loading

The loading icon will replace either the prefix or suffix in that order. If no prefix or suffix is set it will inject itself into the prefix

```tsx live=true
<DeleteMe>
  <Button loading={getState('loading')} prefix={<PlusSVG />}>
    With Prefix
  </Button>
  <Button loading={getState('loading')} suffix={<PlusSVG />}>
    With Suffix
  </Button>
  <Button loading={getState('loading')}>No Prefix or Suffix</Button>
  <Button onClick={() => toggleState('loading')}>Toggle Loading</Button>
</DeleteMe>
```
