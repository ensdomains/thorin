
## Sizes

The size of the avatar is controlled by the width of it's parent component.

```tsx live=true
<Box display="flex" alignItems="center" flexDirection="row" flexWrap="wrap" gap="4">
  <div style={{ width: '100px' }}>
    <Avatar label="Noun 97" src={avatars.noun97} />
  </div>
  <div style={{ width: '75px' }}>
    <Avatar label="Noun 97" src={avatars.noun97} />
  </div>
  <div style={{ width: '50px' }}>
    <Avatar label="Noun 97" src={avatars.noun97} />
  </div>
  <div style={{ width: '25px' }}>
    <Avatar label="Noun 97" src={avatars.noun97} />
  </div>
</Box>
```

## Placeholder

The placeholder is shown if the src property is undefined or if an error occurs loading the src.

```tsx live=true
<Box display="flex" alignItems="center" flexDirection="row" flexWrap="wrap" gap="4">
  <Box display="flex" alignItems="center" gap="4"> 
    <Typography variant="label">undefined src</Typography>
    <div style={{ width: '100px' }}>
      <Avatar />
    </div>
  </Box>
  <Box display="flex" alignItems="center" gap="4">  
    <Typography variant="label">broken Link</Typography>
    <div style={{ width: '100px' }}>
      <Avatar src="http://thisisa.com/brockenLink.jpg" />
    </div>
  </Box>
</Box>
```

## next/image

If you are using Next.js and configured [`next/image`](https://nextjs.org/docs/api-reference/next/image) appropriately, you can pass it in with the `as` prop.

```tsx live=true expand=true
<div style={{width: "50px"}}>
<Avatar as={NextImage} label="Noun 97" src={avatars.noun97} />
</div>
```
