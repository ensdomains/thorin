```tsx
import { Skeleton } from '@ensdomains/thorin'
```

```tsx expand=true live=true
<Box display="flex" flexDirection="column" gap="4">
  <Skeleton loading={getState('skeleton')}>Hello World</Skeleton>
  <VisuallyHidden>{setDefaultState('skeleton', true)}</VisuallyHidden>
  <Button onClick={() => toggleState('skeleton')}>Toggle</Button>
</Box>
```

## Wrap content

The container will automatically resize to encompass wrapped content.

```tsx live=true
<Box display="flex" flexDirection="column" gap="4">
  <Skeleton loading={getState('skeleton')}>
    <Typography>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
      tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim
      veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea
      commodo consequat. Duis aute irure dolor in reprehenderit in voluptate
      velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat
      cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id
      est laborum.
    </Typography>
  </Skeleton>
  <VisuallyHidden>{setDefaultState('skeleton', true)}</VisuallyHidden>
  <Button size="small" onClick={() => toggleState('skeleton')}>
    Toggle Skeletons
  </Button>
</Box>
```

## Skeleton Group

Nest inside [`SkeletonGroup`](/components/SkeletonGroup) to manage multiple skeletons with a single `loading` prop.

```tsx live=true
<Box display="flex" flexDirection="column" gap="4">
  <SkeletonGroup loading={getState('skeleton')}>
    <Skeleton>
      <Heading level="1">Foo Bar Baz</Heading>
    </Skeleton>
    <Skeleton>
      <Typography>The quick brown fox jumped over the lazy dogs.</Typography>
    </Skeleton>
  </SkeletonGroup>
  <VisuallyHidden>{setDefaultState('skeleton', true)}</VisuallyHidden>
  <Button size="small" onClick={() => toggleState('skeleton')}>
    Toggle Skeletons
  </Button>
</Box>
```
